# TDuMAIC AI 互动课堂操作手册

欢迎使用 TDuMAIC AI 互动课堂系统！本手册专门为教师及普通用户等非技术人员编写，帮助您快速上手，构建生动有趣的智能体课堂。

---

## 1. 认识 AI 互动课堂

TDuMAIC 是一个开源的 **AI 互动教学平台**。您只需输入一个教学主题，或上传一份现有的 PDF 课件，系统就能在几分钟内自动为您生成包含演示幻灯片（PPT）、互动测验、3D 虚拟实验以及项目制学习活动（PBL）在内的完整课堂。

进入课堂后，AI 教师将会用语音为您讲解知识，AI 智能体同学会与您一起在课堂中提问、讨论，AI 教师还会在黑板（白板）上绘制图表、书写公式，给您带来完全沉浸式的学习新体验。

---

## 2. 准备工作：配置 AI 服务

在开始生成课程前，您需要为系统配置 AI 智能体的大脑。由于课堂包含文字、语音、图片等多种表现形式，不同功能是由不同的 AI 服务负责的。

### 🧩 AI 服务的职责与依赖关系

在配置前，请先了解不同 AI 服务的“分工”与依赖：

| 模块类型 | 是否必填 | 负责的功能 | 如果未配置会怎样？ |
| :--- | :--- | :--- | :--- |
| **文本大模型** <br>(LLM) | **必填** | 生成课堂大纲、编写幻灯片内容、智能体老师/学生的对话逻辑。 | ❌ 系统将无法生成任何课程，点击“进入课堂”会报错。 |
| **语音合成服务** <br>(TTS) | 选填 | 让 AI 教师和 AI 同学拥有逼真的声音，能够进行语音授课。 | 🔇 课堂变为“静音模式”，智能体仅通过文字气泡交流。 |
| **图像生成模型** <br>(Image) | 选填 | 自动为幻灯片页面绘制精美的插图，丰富视觉效果。 | 🖼️ 幻灯片中需要图片的地方会显示占位图或仅显示文字。 |
| **语音识别服务** <br>(ASR) | 选填 | 允许您在输入框中使用麦克风直接进行语音输入。 | ⌨️ 您只能通过键盘打字输入您的需求和对话。 |
| **视频生成模型** <br>(Video) | 选填 | 用于在课堂中生成精美的演示小视频。 | 🎬 课堂中将无法播放生成的视频内容。 |

```
   ┌────────────────────────────────────────────────────────┐
   │                  文本大模型 (必填·核心大脑)             │
   └───────────┬────────────────────────┬───────────────────┘
               │ (基础生成能力)         │ (多媒体互动增强)
               ▼                        ▼
     ┌──────────────────┐     ┌──────────────────┐
     │ 幻灯片大纲与文字 │     │   语音授课 (TTS) │ ◄── 选填（有声课堂）
     │ 智能体实时对话逻辑│     │   配图生成(Image)│ ◄── 选填（图文课堂）
     └──────────────────┘     │   演示视频(Video)│ ◄── 选填（视频互动）
                              └──────────────────┘
```

> [!IMPORTANT]
> **文本大模型是地基**。您必须先配置至少一个文本大模型服务商（如 Gemini, OpenAI 或通义千问），才能解锁生成幻灯片的基本功能。在此基础上，如果您希望课堂“有声有色”，再对应配置语音合成 (TTS) 和图像生成服务。

### 第一步：打开设置界面
点击主界面右上角的 **齿轮（设置）图标**，即可打开系统配置弹窗。

![LLM 服务商配置界面](/images/llm-settings.png)

### 第二步：获取并填写密钥 (API Key)
系统支持多种模型服务商，您可以选择您已有的服务并填入其密钥（API Key）：
- **Google Gemini (推荐)**：性能与速度的最佳平衡。建议使用 `Gemini 3 Flash`。
- **OpenAI**：支持最新的 GPT 系列模型。
- **DeepSeek / 通义千问 / Kimi**：国内常用的优秀 AI 服务，速度快且成本低。
- **本地 AI (Lemonade)**：如果您是在本地电脑上通过本地 AI 服务运行，则可以实现免密钥一键连接。

> [!TIP]
> 填写好 API 密钥后，请点击“保存”或“确定”。当主界面底部的“进入课堂”按钮变为可点击状态，即代表配置成功！

## 3. 快速上手：一键生成课堂

在主界面中央的输入框中，您可以开启您的第一堂课。

### 方式一：用纯文字描述主题
直接在输入框中写下您想讲授的主题或教学大纲。
- **提示词编写小技巧**：输入越具体，AI 生成的内容质量越高。
  - *普通输入*：“教我量子物理”
  - *优秀输入*：“教我量子物理的基础概念（波粒二象性），适合初中生理解，多举一些生活中的例子，并在课后设计 3 道选择题”

### 方式二：上传 PDF 课件辅助生成
如果您已有做好的课件或参考资料（PDF 格式），可以点击输入框底部的 **“上传 PDF” 按钮** 导入。AI 将会精读您的文档，并以此为大纲和依据来生成幻灯片及教学互动，这对于讲授专业教材非常有用。

### 辅助功能开关
- **网页检索**：如果讲授的主题包含最新时事或前沿技术，请在输入框底部开启 **“网页检索”**。AI 会在生成前自动搜索网络，保证内容的时效性与准确性。
- **深度交互**：开启 **“深度交互”** 按钮。AI 会在课程中穿插 3D 模型观察、模拟互动小游戏或代码在线练习，让学生能够动手实践。
- **语音输入**：如果不方便打字，可以点击输入框右下角的 **麦克风图标**，直接说话将语音转化为文字。

> [!IMPORTANT]
> 准备就绪后，点击右下角带箭头的 **“进入课堂”** 按钮，系统将开始为您规划大纲并逐步构建完整的互动课堂。

## 4. 探索课堂互动模式

课程生成完成后，您将进入沉浸式的学习页面。OpenMAIC 提供了三种独创的学习互动模式：

### 互动幻灯片模式
- **AI 教师主讲**：AI 教师会自动播放幻灯片并进行实时的语音讲解，同时会在屏幕上使用**聚光灯**或**激光笔**高亮核心区域。
- **智能体同学讨论**：讲解过程中，AI 同学（如系统内置的小明等）会随时根据当前内容提出疑问，AI 教师会停下进行解答，您也可以在聊天框中随时打字加入他们的讨论。
- **智能白板**：当涉及到公式推导或结构图时，AI 教师会在右侧的白板上实时绘制图表、书写公式，帮您理清逻辑。

### 随堂测验模式
- **在线答题**：系统会自动在知识点讲完后插入测验卡片，支持单选题、多选题等。
- **即时反馈**：答题后，系统会立即给出正确答案，AI 教师也会针对错误率较高的题目进行语音解析。

### 项目制学习 (PBL) 与 3D 模拟实验
- **动手实践**：在某些互动页面，您可以直接在网页上拖动滑块调节参数，观察 3D 模型或物理化学实验的实时反应。
- **在线编程**：支持直接在网页中编写和运行简易代码，AI 会在一旁为您纠正语法错误。

## 5. 课件保存、导出与分享

您可以将生成的优秀课堂导出保存，方便以后离线使用或分享给他人。

### 导出为 PPTX (幻灯片)
点击课堂右上角的 **“导出 PPTX”** 按钮，AI 生成的幻灯片会被转换成可编辑的 PowerPoint 文件下载。您可以直接使用 Office 或 WPS 软件进行二次修改和美化。

### 导出课堂 ZIP 包 (互动备份)
点击 **“导出课堂 ZIP”**，系统会将整节课的所有数据（包含多智能体脚本、音频文件和白板步骤）打包下载。
- **如何导入**：在主界面，点击“最近课程”列表旁边的 **“导入课堂”** 按钮，选择下载好的 `.maic.zip` 文件，即可一键还原课堂，继续上课。

## 6. 常见问题解答 (FAQ)

### Q1：点击“进入课堂”后，提示“请先配置至少一个模型服务商”？
**答**：这说明您还没有填写 API 密钥。请点击右上角齿轮图标，在对应的服务商处填入正确的 API Key 并保存。

### Q2：生成课程时卡在“生成中”，一直没有反应怎么办？
**答**：生成通常需要 1~3 分钟。如果超过 5 分钟无响应，通常是您的 AI 服务商连接超时间。请检查您的网络环境（是否需要代理），或者在设置中尝试更换其他的 AI 模型。

### Q3：AI 教师的语音太机械，可以换音色吗？
**答**：可以的！在主界面输入框下方的工具栏中，点击“语音合成”图标，您可以选择不同的发音人（男声、女声或童声），甚至可以导入您克隆的专属音色。

### Q4：离线状态下可以使用 OpenMAIC 吗？
**答**：可以。如果您在导出时选择了“导出教学资源包”，它会把幻灯片与离线交互页面打包。对于本地运行的用户，通过配置 `Lemonade` 本地 AI 模型，您可以在完全断网的环境下运行 OpenMAIC 的全部功能。
