## [0.3.0] - 2026-06-25

### Features

- **内置 AI 漫剧课堂案例集成**
  - 在工作台主页首屏默认且置顶展示三门全新的 AI 漫剧推荐课程（*7天AI漫剧基础进阶精品班*、*14天AI漫剧+短剧全流程进阶高手班*、*21天AI漫剧全能创业大师班*）。
  - 实现了**按需延迟下载**与**静默解密/解压导入 IndexedDB** 的无缝初始化流程，在保留极致顺滑体验的同时，极大地降低了客户端首次打开时的网络带宽消耗和内存开销。
  - 引入了基于 `AnimatePresence` 的**高颜值毛玻璃拟态 Loading 进度遮罩层**，分阶段呈现实时导入进度（下载 -> 解析 -> 验证 -> 写入媒体 -> 构建大纲），提供极佳的用户掌控感。
  - 重构并打通了案例展示页（`/cases`）的漫剧入口，点击“进入课堂”可直接触发静默导入并无缝路由跳转至对应课堂详情页。

- **AI 服务配置在线用户手册集成**
  - 将 `public/manuals/AI服务配置详细说明.md` 深度集成到线上，方便终端用户随时查阅。
  - 实现了一个高颜值、体验丝滑的在线用户手册预览抽屉（Drawer/Sheet），用户可在工作台或设置面板中一键拉起、快速翻阅。
- **品牌重塑与推广体系升级**
  - 顺利完成项目向 **TDuMAIC** 的全面品牌重构，清理了代码库中所有历史遗留的 openmaic 引用与陈旧路径。
  - 引入并合成了由 ElevenLabs 渲染的高品质中文旁白产品宣传视频，并将其在 Landing Page 与相关推广渠道上线展示。
  - 优化了 Landing Page 底部资源的在线预览功能。

### Improvements & Optimizations

- **课堂导入核心逻辑解耦**
  - 将原有的上传 ZIP 文件导入流程解耦并提取为独立的底层核心通用函数 `importClassroomCore`，支持指定固定 Stage ID 覆盖随机 nanoid。
  - 使此核心函数能够同时服务于“本地上传导入”与“静态网络 ZIP 静默拉取导入”两种不同的业务流程。
- **TS 类型健壮性优化**
  - 全面修复了内置课堂与用户自建课堂在主页合并、卡片定制渲染、以及异步流式解析中的 TypeScript 编译期类型推导错误，确保生产环境构建（`tsc`）零警告零报错。

### Infrastructure

- **Matt Pocock Skills 规范引入**
  - 梳理并定义了基于 Matt Pocock skills 框架在已有工程上进行二次开发的标准工作流与最佳实践，为后续 AI 编码助手的高效 Pair-Programming 提供了规范化指导。
