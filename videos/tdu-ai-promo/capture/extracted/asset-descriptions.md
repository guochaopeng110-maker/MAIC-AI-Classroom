# 可用画面与素材资产清单 (Asset Inventory)

本清单描述了在设计视频分镜时可以使用的核心插图与动画视觉素材。这些素材可以通过 HyperFrames 原生组件、Canvas、SVG、代码动画或预置媒体直接渲染：

- **`tdu-logo`**: 
  - **描述**: TDu AI 互动课堂的官方横版 LOGO。高清晰度，适合放在开头和结尾作品牌亮相。
  - **对应资源**: `/tdu-logo-horizontal.png` 或 SVG 矢量代码。

- **`scene-class-mockup`**:
  - **描述**: AI 互动课堂授课主界面。左侧为 AI 老师的语音波形与白板板书区，中间为多智能体幻灯片演示区，右侧为 AI 同学讨论群组。
  - **渲染形式**: 纯代码 CSS Grid 结构。

- **`scene-generator-flow`**:
  - **描述**: “一键生成”交互流程。展示在输入框中打字输入“量子物理学入门”或上传 PDF 文档，然后触发一个炫酷的“AI 构建大纲 -> 生成幻灯片 -> 合成课件”的阶段式多任务加载微动画。
  - **渲染形式**: HTML + GSAP 节点动画。

- **`scene-multi-agent`**:
  - **描述**: 多个 AI 智能体在线协作的概念图景。展示不同头像的智能体角色（如 老师、学霸同学、助教）在一个社交网络连接图（Node Graph）中实时交换信息与对话。
  - **渲染形式**: SVG 连线连接发光动画。

- **`scene-interactive-sandbox`**:
  - **描述**: 高交互教学场景。展示一个 3D 物理模拟小实验（如单摆运动控制）或者一套可以实时点击作答并显示正误的测验选择题卡。
  - **渲染形式**: 二维 Canvas 或 SVG 卡片动画。

- **`scene-export-flow`**:
  - **描述**: 一键导出展示。展示点击“导出”按钮，卡片中浮现出 Microsoft PowerPoint 图标（生成可编辑 `.pptx`）和网页 HTML 图标（导出交互式 `.html` 课堂包），并有下载成功的微章。
  - **渲染形式**: CSS 卡片平移与浮现效果。

- **`scene-openclaw-flow`**:
  - **描述**: OpenClaw 聊天机器人集成。展示在飞书 (Lark) / Slack 的对话框界面，用户对 AI 发送指令 “教我量子物理”，AI 智能体直接秒级回复一个卡片并开启互动课堂。
  - **渲染形式**: 仿聊天聊天气泡打字效果。
